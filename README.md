# Api-Rest-SQL
## 🚀  Como usar! </hr>
Instalar a node_Modules:
```
  $ npm install
```
## ⚙️  Packge.json </hr>
Config Nodemon:
```
  "scripts": {
    "dev":"nodemon src/index.js"
  },
```

## 🔥 Start </hr>
Inicia Servidor:
```
  $ npm run dev pra rodar o servidor
```
